# ここにタイトルを入れる

## DEMO

  - https://chat-20231125-4d0ca.web.app/

## 紹介と使い方

  - ログインボタンを押すとグーグル認証が立ち上がり、グーグルアカウントでログイン。その後はチャットが使える。名前の入力は不要で情報はグーグル認証で取得している。
    

## 工夫した点

  -ログインした人のグーグルアカウントの名前とプロフィール画像取得をしその情報をコメントに反映をさせたこと。グーグル認証を導入したこと。

## 苦戦した点
